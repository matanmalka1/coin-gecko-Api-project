Project Description

This is a comprehensive SPA for tracking cryptocurrency prices and trends in real-time, using CoinGecko API and CanvasJS.

Technologies Used
  • HTML5 & CSS3
  • Bootstrap 5
  • jQuery (DOM & AJAX)
  • CanvasJS (charts)
  • CoinGecko API
Features
  • Browse & search (fuzzy) through top coins
  • More info per coin with caching
  • Select up to 5 coins (persisted)
  • Live chart updates every 2s
  • Local caching & graceful errors
